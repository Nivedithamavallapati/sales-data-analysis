# Sales Data Cleaning and Visualization

## Project Overview
This project focuses on cleaning, processing, and visualizing a sales dataset using Python.

## Objectives
- Handle missing values
- Remove duplicates
- Detect and handle outliers
- Create professional visualizations
- Generate business insights and storytelling with data

## Tools and Libraries Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Dataset
Superstore Sales Dataset

## Visualizations Included
- Bar Charts
- Pie Chart
- Heatmap
- Histogram
- Boxplot
- Scatter Plot
- Monthly Sales Trend Dashboard

## Key Findings
- East and West regions generated strong sales.
- Technology category contributed major sales.
- Sales fluctuate across months.
- Outliers were identified and handled in the dataset.

## Conclusion
This project successfully demonstrates data cleaning, visualization, dashboard creation, and storytelling with data using Python.
